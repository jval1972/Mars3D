These are the files from the DOS installation and CD that are needed to run Mars3D.
Some of the files are optional, and a simple warning message may appear if not found. 
Missing files will disable features or may cause crashes.

Minimum File List to play Mars3D (without sounds/animations)
------------------------------------------------------------
MARS.MAD
GAME.PAL
WATER.PAL
MARS.XMI
